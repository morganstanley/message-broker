This static site generator processes MDX `.mdx` in addition to traditional Markdown `.md` files. `MDX` allows for the use of JSX components within Markdown.

Several components based on Morgan Stanley brand guidelines are available to use in `.mdx` files.

## Section

<Example>
  <Section
    category="Category"
    title="Section Title"
    image="http://placekitten.com/800/300"
    color="berry"
  >
    This is a section.
  </Section>
</Example>

<section
  category="Category"
  title="Section Title"
  image="http://placekitten.com/800/300"
